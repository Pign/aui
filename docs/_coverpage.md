# aui

> Build native Android apps in Haxe.

- Reactive state with `@:state`
- 25+ Jetpack Compose views and 30+ modifiers
- Compile-time Kotlin code generation
- Material Design 3 theming
- Tab and stack navigation
- Working CLI: `aui build`, `aui run`
