# Showcase

A 3-tab app demonstrating all widgets and visual effects.

## Tabs

### Widgets tab
- **TextField** with live state binding
- **Counter** with increment, decrement, and reset buttons
- **Toggle** switch for dark mode
- **AlertDialog** triggered by a button

### Modifiers tab
- **Rounded corners** &mdash; `.background(Blue).cornerRadius(8)`
- **Borders** &mdash; `.border(Red, 2).cornerRadius(8)`
- **Shadows** &mdash; `.background(White).shadow()`
- **Opacity** &mdash; `.opacity(0.5)`
- **Scale** &mdash; `.scaleEffect(1.2)`
- **Rotation** &mdash; `.rotationEffect(15)`
- **Fill width** &mdash; `.fillMaxWidth()`
- **Directional padding** &mdash; `.paddingHorizontal(32)`

### About tab
- Framework info with styled text

## What it shows

This example is designed as a visual test of every AUI feature. It uses 5 state variables, 3 tabs, and exercises most modifiers and view types in a single app.
